UnitFix is a lightweight maintenance coordination platform built for landlords and property managers who want a simple, reliable way to manage maintenance requests, assign contractors, and track progress ‚Äî without spreadsheets, texts, or endless email chains.

## Problem

Maintenance coordination is one of the biggest operational pain points in property management:
- Requests come in via texts, calls, and emails
- Contractors miss details or updates
- Landlords have no single source of truth
- Status tracking becomes manual and error-prone

UnitFix solves this by centralizing everything into one clean, easy-to-use system.

## Solution

UnitFix allows landlords to:
- Receive maintenance requests in one place
- Assign requests to contractors
- Track request status from start to finish
- Reduce back-and-forth communication

All without complex setup or bloated features.

---

## Core Features

- **Maintenance Requests**
  - Submit requests with unit info and issue description
  - Optional photo uploads

- **Landlord Dashboard**
  - View all requests in one place
  - Assign contractors
  - Track status: New ‚Üí In Progress ‚Üí Completed

- **Contractor Access**
  - View assigned jobs
  - Update job status
  - Close completed requests

- **Notifications**
  - Email or in-app notifications when status changes
